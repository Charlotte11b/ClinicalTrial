# Example Adverse Event Assessment Script - Replace Before Training Use

Gather factual information without diagnosing:

- Symptom description.
- Onset date/time and timing relative to study medication or procedure.
- Duration and whether it is ongoing.
- Severity using participant-friendly terms.
- Action taken, including medication, urgent care, emergency care, hospitalization, or contacting another clinician.
- Outcome or current status.
- Participant's view of whether it may be related, without leading them.
- Escalate serious, urgent, severe, worsening, or concerning symptoms.
